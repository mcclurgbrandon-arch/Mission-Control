# Daily Memory System

Use this folder for day-by-day logs.

## File naming
- One file per day: `YYYY-MM-DD.md`

## What goes in daily logs
- Important conversations
- Decisions made
- Configuration changes
- Problems encountered and fixes
- Ongoing projects and next steps

## Suggested entry format
```markdown
## HH:MM - Topic
Summary of what happened.

- Key point
- Decision
- Next step
```

## Retrieval workflow
- For questions about prior work, search `MEMORY.md` and `memory/*.md` first.
- Prefer `MEMORY.md` for durable facts and `memory/YYYY-MM-DD.md` for day-specific history.
