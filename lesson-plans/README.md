# Lesson Planning System

This folder is the working area for Brandon's lesson planning.

## Structure
- `templates/` — reusable lesson/unit templates
- `current-unit/` — active unit materials being planned now
- `weekly/` — week-by-week plans
- `assessments/` — quizzes, tests, exit tickets, rubrics
- `labs/` — lab plans, setups, handouts
- `archive/` — older completed materials

## Workflow
1. Put source materials here (exported from Google Drive or copied in).
2. Draft or revise plans in `current-unit/` or `weekly/`.
3. Store reusable formats in `templates/`.
4. Move finished materials into `archive/` as needed.

## Naming suggestions
- Weekly plans: `YYYY-W##-topic.md`
- Daily plans: `YYYY-MM-DD-topic.md`
- Unit overviews: `unit-name-overview.md`

## Working pattern with TARS
- Ask for a new lesson, revision, pacing adjustment, quiz, lab, or differentiation.
- TARS can draft in Markdown first, then adapt to another format if needed.
- Keep key source materials here so TARS can refer back to them.
